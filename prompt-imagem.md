# Prompt para OpenAI Image Generation (DALL-E 3)

## Use este prompt no ChatGPT ou na API da OpenAI

> Create a modern, professional hero image for a GitHub README — a Portuguese Food & Wine RAG (Retrieval-Augmented Generation) guide built for LLM Zoomcamp 2026.
>
> ## Purpose
> A single striking image that communicates: Portuguese cuisine + AI-powered RAG system. To be used as the cover image for a GitHub project README.
>
> ## Style Guidelines
> - Clean, premium, food-tech aesthetic — warm and inviting but professional.
> - Color scheme: Warm burgundy #722F37 (primary), gold #C4975A (accent), cream #FDF8F3 (background), dark charcoal #3D3033 (text).
> - Visual style: Modern flat design with subtle gradients, geometric shapes, and minimal icons.
> - Typography: Clean sans-serif for body, elegant serif for headings.
> - Format: 1200 x 630 px (horizontal, optimized for GitHub README).
>
> ## Main Visual Elements
> - Center: A stylized wine glass and a traditional Portuguese dish (like Pastel de Nata or a tasca table) rendered as modern flat icons.
> - From the dish/glass, a subtle glowing "data stream" flows upward into an abstract cloud/AI node — representing the RAG pipeline.
> - Small floating icons around the AI node: a magnifying glass (search), a document (knowledge base), a chat bubble (conversation), a chart (monitoring).
>
> ## Text Overlay
> - Title (large, bold serif): "Portuguese Food & Wine Guide"
> - Subtitle (smaller, sans-serif): "RAG Assistant powered by LLM Zoomcamp 2026"
> - Tagline at bottom: "Pergunte sobre comida e vinho portugueses"
>
> ## Key Concepts (displayed as small badge/pill labels floating in the design)
> - "Hybrid Search"
> - "GPT-4o"
> - "Wine Regions"
> - "Traditional Recipes"
> - "Citations"
> - "Real-time Monitoring"
>
> ## Bottom Bar
> - A thin decorative bar with small icons representing the tech stack: Flask, Qdrant, PostgreSQL, Grafana, Docker.
> - Small text: "LLM Zoomcamp 2026 — Final Project"
>
> ## Output Requirements
> - Output a single 1200x630 PNG image.
> - Keep text concise and high contrast.
> - Ensure all elements are legible at README thumbnail size.
> - Do not include any human faces or people.
> - No text other than what is specified above.
